# 🚀 TaskFlow – Full Stack MERN To-Do Application

TaskFlow is a modern and responsive task management application developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The application provides secure user authentication and complete task management functionality through an intuitive and user-friendly interface.

## ✨ Key Features

* Secure User Registration and Login
* JWT-based Authentication
* Protected Routes for Authorized Access
* Create, Read, Update, and Delete (CRUD) Tasks
* Mark Tasks as Completed
* Dynamic Dashboard Statistics
* Logout Functionality
* Responsive and Modern User Interface
* Seamless Integration with MongoDB Atlas

## 🛠️ Technology Stack

### Frontend

* React.js
* React Router DOM
* Axios
* CSS

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JSON Web Token (JWT)
* bcryptjs

## 📌 Core Functionalities

* User Authentication with Password Encryption
